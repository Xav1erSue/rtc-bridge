# RTC Bridge

## Introduction

this is a simple video stream processing project, based on `Python` and `GStreamer`.

the core implements of stream processing is forked from [gstreamer-webrtc](https://github.com/notedit/gstreamer-webrtc)

## Features

- [x] Signaling Server

- [x] HTTP Server

- [x] WebRTC to RTSP

- [x] RTSP to WebRTC
